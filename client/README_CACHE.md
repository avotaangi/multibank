# Решение проблемы с кешированием в Telegram Mini App

## Проблема
Изменения не отображаются в Telegram Mini App из-за агрессивного кеширования.

## Решения

### 1. Очистка кеша в Telegram
- Откройте Telegram Mini App
- Зажмите и удерживайте кнопку "Обновить" (если есть)
- Или закройте и снова откройте приложение

### 2. Очистка кеша браузера
- В Telegram: Настройки → Данные и память → Очистить кеш
- Или перезапустите Telegram

### 3. Принудительное обновление
- Добавьте параметр `?v=timestamp` к URL приложения
- Или используйте жесткую перезагрузку (Ctrl+Shift+R или Cmd+Shift+R)

### 4. Разработка
- В dev режиме Vite автоматически добавляет параметры версии
- При каждом изменении кода происходит hot reload
- Если не работает - перезапустите dev server: `npm run dev`

### 5. Production
- После сборки (`npm run build`) изменения должны быть видны
- Если нет - очистите кеш браузера/Telegram
- Или измените версию в package.json

## Технические детали

В коде реализовано:
1. Автоматическая очистка Service Worker кешей
2. Проверка версии приложения при загрузке
3. Принудительное обновление при изменении версии
4. Cache-Control headers для предотвращения кеширования
